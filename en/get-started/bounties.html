Development bounties

Veil is open source. If you want to help, jump in — pick a task, improve something, and ship. Many contributions start small and grow from there.

How bounties work today: some issues are tagged as bounties and may include a listed reward. We’re actively refreshing the bounty list to match current priorities, so you may see a mix of older and newer items. If you’re unsure whether something is still relevant, ask — we’ll confirm and help scope it.

How to claim or coordinate work

  • Start on GitHub — Open the issue and leave a comment saying you want to work on it (or ask clarifying questions).
  • Coordinate in the main Discord — If you want faster back-and-forth, drop a message in Discord and link the issue.
  • Submit your work — Open a pull request referencing the issue.
  • Bounty payout — If the issue is funded, we confirm the reward amount and pay once the work is accepted.
