Current version is 1.4.0.0 a mandatory upgrade. The previous mandatory wallet released was version 1.3.0.0. Wallet issues are tracked at GitHub Issues List. Daily blockchain snapshots are available for download at the Veil Tools website

Windows Linux macOS

Community-written setup guides are here (please note that formatting and navigation may differ from the main website). YouTube versions and blog posts are encouraged:

  • How to set up the Veil wallet in Ubuntu Linux (GUI/Qt wallet).
  • How to set up the Veil wallet in macOS (GUI/Qt wallet).

Additionally, there is a paper wallet generator for both the mainnet and testnet. While a paper wallet generator can run online and offline, as a privacy-centric project, our recommendation is to download and run locally. After paper wallet user feedback we will give this a more prominent position on this page.

Future wallets planned include desktop and mobile light wallets and support for hardware wallets—such as Ledger and Trezor.

Following integration with hardware wallets, work on delegated staking is envisaged, allowing Veil users to stake their coins from the security of a hardware wallet.
